## Installation

### CDN

```html
<link rel="stylesheet" href="dist/bentou.css">
```

### Manual

Download `bentou.css` from the `dist` folder and include it in your HTML:

```html
<link rel="stylesheet" href="path/to/bentou.css">
```

## Usage

Simply include the CSS file in your HTML and start using the classes:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="bentou.css">
  <title>My App</title>
</head>
<body>
  <div class="container">
    <h1>Hello, Bentou!</h1>
    <button class="btn btn-primary">Get Started</button>
  </div>
</body>
</html>
```
